# GOF patterns
> This application shows GOF patterns implementation using modern C++
<hr>

# How to build:
```
git clone --recurse-submodules git@github.com:korenandr/gof_patterns.git
cd ./gof_patterns && cmake -S . -B build
cmake --build build
```

# How to use (example):
```
cd ./build/src
./gof_patterns --help
./gof_patterns --pattern memento
```
